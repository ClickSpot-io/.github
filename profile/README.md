# ClickSpot

Link analytics platform for marketers. Track every click, prove every campaign.

## Features

- **Shareable Analytics** — Public links to share click data with clients
- **UTM Tracking** — Built-in campaign parameter builder
- **Geographic Insights** — Country, city, region breakdowns
- **Custom Domains** — Unlimited
- **Unlimited QR Codes** — High-resolution for print campaigns
- **Team Workspaces** — No per-seat pricing
